I am very much intrigued with reinforced learning. A personal project of making bots in a player-vs-player (playing against other players) and player-vs-environment (usually incorporate min-maxing) settings in video games drove me to write successful algorithms. Some optimizations include, as you might imagine, navigation (optimizing the shortest distance to a moving object) shortest distance around moving objects from A to B (imagine moving spheres, what is the shortest distance to B as the sphere objects move dynamically?), min-maxing against other players (what action would lessen the highest possible success in the opponent?). These are all common problems that Machine learning can be used, and video games are a fantastic way to learn how to code these.
